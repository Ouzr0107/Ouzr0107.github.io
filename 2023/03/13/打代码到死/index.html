打代码到死

2023-03-13
#学校
代码

打了一天的代码了，感觉自己好像什么都没干，有一种自闭的空虚感，好希望可以看到成品。可能，可以马上看到成果就是前端动力的来源吧！后端就一直在写接口，虽然说 API 不用每个字都手打（因为我用了 Apifox），但是就没有什么正反馈，只有碰到 BUG 的负反馈。

啊啊啊！虽然我之前说不想买显示器了，但其实一个大的显示器好像可以让屏幕更近一些，毕竟笔记本屏幕显示的细腻程度已经是我能接受的水平了。我的眼睛在经过了寒假每天高浓度的玩耍和熬夜之后，好像度数有点加深了，上课坐后排看PPT开始有些模糊和吃力。难道说，我可以换眼镜了？其实我想换一个只有上边框的眼镜挺久了，看我动森的形象就能知道 2333 。

就这样吧，一点牢骚罢了。
